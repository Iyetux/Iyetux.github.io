vue init 一直都是downloading template

نُشر في 2020-05-31 عُدل في 2024-08-26

在使用 vue init 「template-name」 时可能会一直停留在downloading template造成这种现象的原因，只能归属到github，或者npm下载timeout或者网络慢的原因。

  1. 此时可以尝试直接下载「template-name」压缩包。
  2. 解压并存储好「template-name」解压路径
  3. 使用 vue init 「template-name」路径 「新建项目名称」的格式
    以上不步骤可以直接解决长时间downloading template问题。
