vuex.a.store is not a constructor

نُشر في 2020-06-02 عُدل في 2024-08-26

控制台报错TypeError: WEBPACK_IMPORTED_MODULE_1_vuex.a.store is not a constructor

这个报错的是_vuex2.default.store 不是一个构造函数因为在我们用vuex的时候需要将用到的actions，mutations模块最终导出，在导出的时候new Vuex.Store中的Store小写了，这里的一定要大写,就相当于我们在使用构造函数(类)的时候首字母要大写

错误案例

正确案例

# vuex store
vue init 一直都是downloading template
hexo deploy Error: Spawn failed
